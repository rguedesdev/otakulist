Este projeto é uma tentativa de criar um trancker de animes, mangas e light novels mais organizado e com mais conteúdo.

## ✨ Features

### ✅ Implementadas

- Página de Anime (ainda em desenvolvimento)

### 🔜 Em desenvolvimento

- Autenticação com JWT
- Cadastro e login de usuários (Backend/Frontend)
- Integração com MongoDB para persistência de dados
- Edição de Perfil
- Adicionar amigos

## 💻 Tecnologias utilizadas

### 🔹 Backend

- Node.js (Runtime)
- Fastify (Framework)
- TypeScript/JavaScript
- JWT (Autenticação)
- MongoDB (Banco de Dados NoSql)
- Mongoose (ORM)
- GraphQl (Requisições)

### 🔹 Frontend

- Next.js com TypeScript
- CSS nativo

### 🔹 Infraestrutura

- Docker (containerização)
